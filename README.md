# 1.PS Podskaláci - Website

This repository contains the source code for the official website of the 1. pioneer group Podskaláci.

## Technologies

- **HTML5**: Website structure
- **CSS3**: Styling and responsiveness (custom styles)
- **JavaScript**: Simple scripts for dark mode and mobile menu

## Project Structure

- `index.html` - Main website page.
- `assets/`
  - `css/` - Cascading stylesheets (main style `styles.css`).
  - `images/` - Images and logos used on the website.

## Running the Project (Local Development)

The website was built with an emphasis on simplicity and does not contain any complex backend code. For local viewing, you can simply open the `index.html` file in your favorite web browser.

To better simulate a production environment, we recommend running the project using a simple local server. For example, using Python from the command line (in the project folder):

```bash
python -m http.server
```
Then open the address `http://localhost:8000` in your browser.
